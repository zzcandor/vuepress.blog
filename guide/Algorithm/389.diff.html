# 找不同

leetcode地址：389.找不同 (opens new window)

# 题目

  • 给定两个字符串 s 和 t，它们只包含小写字母。字符串 t 由字符串 s 随机重排，然后在随机位置添加一个字母。请找出在 t 中被添加的字母。
  • 示例1：
    输入：s = "abcd", t = "abcde"
输出："e"
解释：'e' 是那个被添加的字母。
  • 示例2：
    输入：s = "", t = "y"
输出："y"

# 解法

  1. 笨笨的我用了两个哈希表来储存字符串，然后再对比不同。看了别人的题解，感觉自己是🐖。
var findTheDifference = function(s, t) {
  let maps = new Map()
  let mapt = new Map()
  for(let i of s){
      if(maps.has(i)){
          maps.set(i, maps.get(i)+1)
      }else{
          maps.set(i,0)
      }
  }
  for(let i of t){
      if(mapt.has(i)){
          mapt.set(i, mapt.get(i)+1)
      }else{
          mapt.set(i,0)
      }
  }
  for(let [key, value] of mapt.entries()){
      if(!(maps.has(key)&&maps.get(key) === value)){
          return key
      }
  }
};
  1. 计数，遍历s，将每个字符的数量加一，然后遍历t，再将每个对应数字减一，如果数量为-1，则说明是要找的字符。
  2. 求和，将每个字符的ASCII码加起来，然后两个字符串相减，结果就是答案对应的ASCII码。
  3. 位运算，将两个字符串合并，问题就变成求字符串中出现次数为奇数的字符，对每个字符的ASCII码进行异或运算，剩下的就是答案。
    var findTheDifference = function(s, t) {
  let c = 0
  for(let char of s+t){
      c ^= char.charCodeAt()
  }
  return String.fromCharCode(c)
};
